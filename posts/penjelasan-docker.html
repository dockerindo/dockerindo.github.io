Penjelasan Docker

Mengenal lebih jauh, mulai dari penjelasannya.

Sebelumnya dockerindo sudah membahas Memulai belajar docker dengan Katakoda. Mungkin sudah ada yang mencobanya, tetapi masih belum paham apa itu sebenarnya Docker.
Seperti pepatah saja "Tak kenal maka tak sayang", Dan begitu juga ketika kalian mempelajari docker, kalian tidak akan bisa sayang jika belum mengenal docker itu sendiri.

Docker adalah sebuah project open-source yang diperuntukan oleh para developer dan sysadmin, untuk membuat, mengemas, dan menjalankan aplikasi sebagai container yang ringan.
pasti akan muncul lagi pertanyaan apa itu container? sama seperti saya sewaktu pertama mengenal docker itu sendiri, Container bisa diibaratkan dengan sebuah VM, yang menjalankan Sebuah sistem operasi pada OS host, tetapi perbedaannya docker adalah jika pada virtualisasi berjalan diatas Hypervisor (Aplikasi pihak ke3), docker berjalan pada OS host itu sendiri, dan container juga lebih ringan dan kecil ukurannya.
Jadi dapat disimpulkan kalau container adalah wadah ringan yang digunakan untuk mengemas aplikasi untuk dijalankan.

Satu lagi, docker dikembangkan oleh Solomon Hykes sebagai project internal di dotCloud bersama dengan parter lainnya.
